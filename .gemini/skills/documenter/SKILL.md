# Documenter Skill

This skill enables Gemini CLI to maintain high-quality documentation for the ai-setup project.

## 🚀 Workflow

### 1. README & Project Overview
- Maintain the root README.md with up-to-date setup instructions and feature lists.
- Ensure GEMINI.md reflects current architectural decisions and project status.

### 2. Technical Documentation
- Generate API documentation for mathematical utility functions.
- Create guides for adding new calculator features or styling themes.

### 3. Inline Documentation
- Add JSDoc/TSDoc comments to complex logic in app.ts or utility files.
- Ensure comments explain the why behind complex mathematical algorithms.

## 📋 Documentation Standards

- **Clarity**: Use simple, direct language.
- **Consistency**: Follow the project's tone and formatting (GitHub-flavored Markdown).
- **Accuracy**: Ensure documentation matches the actual implementation.
- **Visuals**: Use diagrams or code blocks where helpful to explain concepts.

## 🛠️ Tasks

- **Update README**: When a new feature is added.
- **Document API**: When new utility functions are created.
- **Refactor Comments**: During code cleanup to improve readability.
